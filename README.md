# NovaCart - E-Commerce Full Stack Web Application

A full-stack e-commerce web application built using React JS, Spring Boot, Spring Security, JWT Authentication, MySQL, and Razorpay.

## 🚀 Live Demo

**Frontend:** https://novacart-ten.vercel.app/

**Backend API:**https://ecommerce-major-project-production.up.railway.app/products

## 🚀 Technologies Used

### Frontend
- React JS
- React Router
- Axios
- Bootstrap
- CSS

### Backend
- Spring Boot
- Spring Security
- JWT Authentication
- Spring Data JPA
- Maven

### Database
- MySQL

### Payment Gateway
- Razorpay

## ✨ Features

## User Features
- User Registration & Login
- JWT Authentication
- Browse Products
- Product Search
- Category Filtering
- Product Details
- Shopping Cart
- Secure Online Payments with Razorpay
- Order Placement
- My Orders

## Admin Features
- Admin Dashboard
- Add Products
- Edit Products
- Delete Products
- View Customer Orders

## 🌐 Deployment

- Frontend: Vercel
- Backend: Railway
- Database: MySQL (Railway)

## ⚙️ How to Run

### Backend

```bash
cd backend
./mvnw spring-boot:run
```

### Frontend

```bash
cd frontend
npm install
npm run dev
```

### Database

Import the `database/ecommerce.sql` file into MySQL.
