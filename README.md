# NovaCart - E-Commerce Full Stack Web Application

A full-stack e-commerce web application built using React JS, Spring Boot, Spring Security, JWT Authentication, MySQL, and Razorpay.

## 🚀 Technologies Used

### Frontend
- React JS
- React Router
- Axios
- Bootstrap
- CSS

### Backend
- Spring Boot
- Spring Security
- JWT Authentication
- Spring Data JPA
- Maven

### Database
- MySQL

### Payment Gateway
- Razorpay

## ✨ Features

- User Registration
- User Login with JWT Authentication
- Product Listing
- Product Categories
- Shopping Cart
- Online Payment using Razorpay
- Order Placement
- My Orders Page

## ⚙️ How to Run

### Backend

```bash
cd backend
./mvnw spring-boot:run
```

### Frontend

```bash
cd frontend
npm install
npm run dev
```

### Database

Import the `database/ecommerce.sql` file into MySQL.
